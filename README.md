# CBE LearnMate

An interactive, child-friendly educational app for the Kenyan Competency Based Education system.

## Features
- **Daily Lessons:** Structured learning path.
- **Progress Tracking:** Gamified stats and levels.
- **Skills Portfolio:** Upload evidence of work.
- **Motivation:** Mascots and celebrations.
- **Responsive Design:** Mobile-first, fast, and accessible.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open in browser:
   `http://localhost:5173` (or the port shown in terminal)

## Technologies
- React (Vite)
- Vanilla CSS (Custom properties, responsive layout)
- Lucide React Icons
- React Router DOM
